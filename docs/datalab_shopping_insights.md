# 쇼핑인사이트 API 가이드

쇼핑인사이트 API는 네이버 데이터랩의 쇼핑인사이트를 API로 실행할 수 있게 하는 RESTful API입니다.
네이버 통합검색의 쇼핑 영역과 네이버쇼핑에서의 검색 클릭 추이 데이터를 JSON 형식으로 반환합니다.

## API 개요
- **비로그인 방식 오픈 API**: 클라이언트 아이디와 시크릿 값만 헤더에 추가하여 호출.
- 하루 호출 한도는 1,000회.

## 쇼핑인사이트 API 레퍼런스
### 1. 분야별 트렌드 조회
쇼핑 분야의 검색 클릭 추이를 조회.
- **URL**: `https://openapi.naver.com/v1/datalab/shopping/categories`
- **메서드**: POST
- **파라미터 (JSON)**: `startDate`, `endDate`, `timeUnit` 등.
- **요청 예**:
```json
{
  "startDate": "2017-08-01",
  "endDate": "2017-09-30",
  "timeUnit": "month",
  "category": [
      {"name": "패션의류", "param": [ "50000000"]},
      {"name": "화장품/미용", "param": [ "50000002"]}
  ],
  "device": "pc",
  "gender": "f",
  "ages": [ "20", "30"]
}
```

### 2. 분야 내 기기별, 성별, 연령별 트렌드
특정 카테고리(분야)에 대해 기기별, 성별, 연령별로 트렌드를 확인할 수 있습니다.

### 3. 키워드 분야/기기별/성별/연령별 트렌드
특정 분야 내에서 특정 키워드에 대한 상세 트렌드 조회가 가능합니다.
- `cat_id`와 `keyword` (키워드/단어) 파라미터를 사용.
- **응답 예시 (연령별 트렌드)**:
```json
{ "startDate": "2017-08-01",
  "endDate": "2017-09-30",
  "timeUnit": "month",
  "results": [
    { "title": "정장", "keyword": ["정장"], 
      "data": [ 
        { "period": "2017-08-01", "group": "10", "ratio": 9.7021 },
        { "period": "2017-08-01", "group": "20", "ratio": 57.88466 }
      ]
    }
  ]
}
```

## 주요 오류 코드
- **403 오류**: 등록한 애플리케이션 설정에서 "데이터랩 (쇼핑인사이트)" 기능이 선택되어 있는지 확인 필요.
