# API 공통 가이드

네이버 오픈API는 외부 개발자가 플랫폼의 기능을 쉽게 이용할 수 있게 웹/SDK의 형태로 공개된 기술입니다.

## 가이드 구성
1. **네이버 오픈API 종류**: 로그인 방식 / 비로그인 방식 조회
2. **사전 준비 사항**: 애플리케이션 등록, 세부 정보 설정, 클라이언트 ID/Secret 확인
3. **내 애플리케이션 관리**: 기본 정보, API 설정, 멤버 관리, 통계 보기, 애플리케이션 삭제
4. **용어 정리**: 
   - API의 기본
   - API 인증
   - API 호출 방식
   - HTTP
5. **샘플 코드**: 로그인 방식 / 비로그인 방식 API 호출 예제
6. **오류 코드**: 주요 오류 코드 및 메시지 형식 안내
