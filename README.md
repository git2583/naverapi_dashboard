# Naver API Dashboard Documentation

이 저장소는 네이버 오픈 API (데이터랩, 로컬/검색 쇼핑, 공통 등) 사용을 위한 개발 가이드라인과 레퍼런스를 정리한 마크다운 문서들로 구성되어 있습니다.

## 포함된 문서
- `docs/datalab.md`: 데이터랩 및 통합검색어 추이 개요
- `docs/datalab_shopping_insights.md`: 쇼핑인사이트 검색, 분야, 키워드 트렌드 조회
- `docs/search_shopping.md`: 쇼핑 검색 API 구현 가이드
- `docs/apilist.md`: 로그인 및 비로그인 기반의 네이버 API 목록
- `docs/openapi_guide.md`: 오픈 API 활용을 위한 공통 사항 안내
- `docs/summary.md`: 문서 수집 과정 요약본
